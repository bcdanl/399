Q3c.

What plots does the following code make? What does . do?

ggplot(data = mpg) + 
  geom_point(mapping = aes(x = displ, y = hwy),
             alpha = .5) +
  facet_grid(drv ~ .)

ggplot(data = mpg) + 
  geom_point(mapping = aes(x = displ, y = hwy),
             alpha = .5) +
  facet_grid(drv ~ .)

  geom_point(mapping = aes(x = displ, y = hwy),
             alpha = .5) +
  facet_grid(. ~ cyl)
